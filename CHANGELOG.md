# 版本变更记录

本项目采用语义化版本号。每个开发阶段完成浏览器验证后，都会提交代码、创建版本标签并推送远程仓库。

## [0.1.0] - 2026-07-20

### 新增

- 使用 Vite 创建 Vue 3 项目脚手架。
- 接入 Pinia 全局状态管理。
- 接入 localStorage 状态持久化示例。
- 创建响应式 Canvas 游戏舞台。
- 添加全屏快捷键、`render_game_to_text` 和 `advanceTime`。
- 完成纯前端竖屏霓虹打砖块的完整开发文档。
- 定义 `v0.1.0` 至 `v1.0.0` 的阶段路线和验收标准。

### 验证

- `pnpm build` 构建通过。
- Playwright 点击 Pinia 测试按钮通过。
- Canvas 截图检查通过。
- 用户现有 Chrome 的完整桌面与 390 × 844 手机视口 UI 回归通过。
- localStorage 刷新恢复验证通过。
- 浏览器控制台无错误。
