# 版本变更记录

本项目采用语义化版本号。每个开发阶段完成浏览器验证后，都会提交代码、创建版本标签并推送远程仓库。

## [0.3.0] - 2026-07-20

### 新增

- 新增多球分裂、挡板扩展、穿透光球、时流减速、双轨激光 5 种正面道具。
- 新增最多 12 球的多球循环；计时道具支持同时生效、同类刷新和扩展挡板两层叠加。
- 新增发光胶囊掉落、挡板接取、漏接消失，以及掉球时统一清空临时效果。
- 新增晶币散落、重力、一次弹跳、近挡板磁吸、接取计数与本地持久化。
- 新增双轨激光主动射击和 240ms 射击冷却。
- 新增 2.4 秒连击窗口、分数倍率、最高连击和 HUD 连击进度。
- 新增 12/25ms 命中停顿、扩散波纹、浮动分数、屏幕闪光和增强震屏。
- 新增粒子对象池；高特效上限 600、低特效上限 240。
- 桌面端新增晶币、连击、活动模块列表与实时倒计时。
- 手机端新增晶币、连击、活动模块胶囊和激光主操作按钮。
- 扩充文本测试状态，包含掉落物、弹体、活动道具、晶币、连击和粒子池。
- 新增可重复调用的道具、掉落物、掉球和清关调试入口。
- 新增 [v0.3.0 验收报告](./docs/v0.3.0验收报告.md)。

### 修复

- 多球逻辑由单球索引改为完整球列表，最后一球落底时才扣除生命。
- 修复多个计时道具同时激活时，390×844 和 360×800 底部按钮被挤出视口的问题。
- 激光激活时保留独立暂停入口，避免主按钮职责冲突。

### 验证

- 生产构建通过，浏览器控制台无错误。
- 5 种道具、12 球上限、两层挡板扩展、计时刷新与掉球清空通过。
- 减速位移倍率实测 0.72；穿透球击穿后纵向速度不反转。
- 双轨激光单次发射生成 2 条弹体，并命中 2 点砖块耐久。
- 晶币磁吸接取、连击递增、奖励负载和粒子上限通过。
- 自动挡板在 3739 帧、31.2 秒内清空 51 块砖，3 条生命通关，获得 41 晶币。
- Chrome 桌面实际开始/发球通过；390×844 与 360×800 竖屏 UI 通过。
- 4 个计时道具同时显示时，两档手机视口主操作按钮完全可见且无横向溢出。

## [0.2.0] - 2026-07-20

### 新增

- 将模板 Canvas 替换为 `540 × 960` 竖屏霓虹游戏战场。
- 新增独立 `GameEngine`，使用 `1/120` 秒固定时间步驱动物理。
- 新增第 1 关“初次折射”的数据驱动砖阵，共 51 块砖。
- 新增普通砖和 2 点耐久强化砖，使用颜色、边框和耐久点区分。
- 新增挡板、发球、小球拖尾、墙面碰撞、挡板角度反弹和砖块碰撞。
- 新增 3 条生命、掉球重置、失败、清关和重新挑战流程。
- 新增键盘、鼠标和触摸控制。
- 新增暂停/继续与全屏操作。
- 新增基础粒子、砖块闪光、球体辉光、震屏和终局画面。
- 新增桌面任务面板、操作面板、进度、生命和状态 HUD。
- 新增手机端单屏战场与底部主操作栏。
- 扩充 `render_game_to_text`，输出挡板、小球、砖块、生命、得分和特效状态。
- 新增 [v0.2.0 验收报告](./docs/v0.2.0验收报告.md)。

### 修复

- 修复小球与强化砖连续重叠时，同一次接触重复扣除耐久和重复计分的问题。
- 增加碰撞位置修正与同砖短暂重复命中保护。
- 清理引擎销毁时遗留的 Canvas 右键菜单监听器。

### 验证

- 生产构建通过。
- 菜单、准备、发球、首次破砖和计分的 Playwright 流程通过。
- 挡板左右边界、挡板反弹、暂停冻结与恢复通过。
- 强化砖一次接触只扣除 1 点耐久。
- 连续扣除 3 条生命后正确进入失败。
- 自动挡板完成 51 块砖的完整清关，最终得分 10260。
- 触摸拖动至左右边界和触摸发球通过。
- Chrome 桌面、1280×720、390×844 和 360×800 UI 回归通过。
- Chrome 完整刷新后无新增控制台警告或错误。

- localStorage 刷新恢复验证通过。
- 浏览器控制台无错误。
