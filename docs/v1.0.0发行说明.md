# 霓虹破界：Neon Breaker v1.0.0 发行说明

发行日期：2026-07-20
类型：纯前端单机竖屏街机游戏
技术栈：Vue 3 + Pinia + Vite + Canvas 2D

## 正式名称

游戏正式中文名为 **霓虹破界**，英文名保留 **Neon Breaker**。

“破界”同时指向挡板突破砖阵、四章核心边界和无尽磁域的持续推进。正式图标由霓虹核心、小球与挡板组成，不使用第三方游戏素材。

## 正式版内容

- 四章 20 个手工主线关卡。
- 第 5、10、15、20 关四个独立 Boss。
- 普通砖、2/3 点强化砖、移动砖、护盾砖和熔芯机关砖。
- 多球、挡板扩展、穿透光球、时流减速、双轨激光五种正面道具。
- 晶币掉落、八档星级奖励和七类永久升级。
- 通关第 10 关后解锁的无尽磁域。
- 键盘、鼠标与触摸操作，竖屏手机和桌面三栏布局。
- SAVE V3 本地档案、旧档迁移、损坏恢复和存储失败降级。
- 自动/高/中/低画面选择、震屏开关和减弱闪光。
- 全程无背景音乐和音效，反馈完全由画面表达。

## 正式版体验收口

- 品牌标题、浏览器标签、应用清单与图标统一为“霓虹破界”。
- 构建基础路径改为相对地址，可部署在域名根目录或任意子目录。
- 新增一键 `pnpm test:release` 发行门禁。
- 新增目标范围审计，直接核验 20 关、5 道具、7 升级、4 Boss、无尽、SAVE V3、品牌资产和双端样式。
- 新档自动模拟顺序完成 20 关、取得 60 星并结算无尽纪录，作为发行证据。

## 存档兼容

正式版继续使用 `saveVersion = 3`，不会重置 `v0.9.0` 档案。v1/v2 正式档案和更早的 Pinia 快照仍可自动迁移。

升级前建议保持同一域名和浏览器。域名变化会进入新的 localStorage 空间。

## 已知设计边界

- 游戏刻意不包含任何音频。
- 档案只保存在当前浏览器，没有云同步。
- 自动画质依据浏览器报告的能力选择，玩家可以在设置页固定档位。
- 无尽模式没有最终波次，以最高分、最高波次和最佳连击作为长期目标。
