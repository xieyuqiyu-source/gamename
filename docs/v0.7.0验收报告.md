## 3. MAGNETRON IX Boss

### 3.1 核心与模块

- 核心耐久：15。
- 阶段：3，每阶段 5 点核心耐久。
- 核心速度：`108 / 148 / 194` 逻辑像素每秒。
- 三阶段分别使用独立磁盾阵列。
- 左右攻击模块：各 3 点耐久。
- 阵列护盾在线时联锁保护模块；清空节点后模块才可受伤并开始齐射。
- 模块射击间隔：`3.8 / 2.8 / 2.1` 秒。

### 3.2 战斗链路

```text
清空阶段磁盾
  → 攻击模块解除联锁并进入齐射窗口
  → 在线模块周期发射磁暴脉冲
  → 脉冲命中挡板
  → 有永久护盾：消耗 1 层并继续战斗
  → 无永久护盾：扣除 1 条生命并重新发球
  → 已购买护盾升级：生命重置后恢复 1 层

阵列清空后核心同步暴露
  → 每损失 5 点耐久进入下一阶段
  → 两座攻击模块未全毁时锁定最后 1 点耐久
  → 模块全毁且核心归零
  → 正式胜利
```

小球和双轨激光都可以伤害攻击模块。模块摧毁后不会随核心阶段重构复活；全部模块离线时会清除场上残余脉冲并永久停止齐射。

### 3.3 视觉与 HUD

- Canvas 核心两侧绘制独立炮塔、耐久条、受击闪白和摧毁衰减状态。
- 磁暴脉冲使用蓝紫菱形、辉光和轻微摆动，与玩家球和掉落物明显区分。
- Canvas 顶部 Boss 行增加在线模块数量。
- 桌面 Boss 终端增加 `ATTACK MODULES` 和场上 `PULSE` 数量。
- 手机紧凑状态行优先显示当前 Boss 阶段和模块数量，不新增纵向卡片。

## 4. 无尽磁域

### 4.1 解锁与入口

- 主线第 10 关胜利时将 `endless.unlocked` 写入正式存档。
- 战役大厅新增独立无尽入口；锁定状态显示解锁条件，解锁状态显示最高分、最高波次和最佳连击。
- 无尽入口与星级奖励在桌面并排、手机纵向紧凑排列。

### 4.2 波次成长

- 每局从第 1 波开始。
- 清空砖阵后直接进入下一波，生命、分数、晶币、连击、未过期掉落物和即时道具不重置。
- 球速每波增加 `0.025`，最高倍率 `1.46`。
- 存活球同步到新波次基础速度且不会被降速，使其与掉命后的重新发球速度遵循同一份波次配置。
- 第 3 波起出现移动砖行；第 4 波起增加 2 点砖；第 9 波起出现 3 点耐久砖。
- 每清一波额外增加 `500 × 已完成波次` 分数。

### 4.3 纪录

失败结算保存：

- 历史最高分。
- 抵达最高波次。
- 历史最佳连击。
- 本局获得的晶币。

纪录采用最大值合并，不会被较差的新成绩覆盖；晶币按当前正式总量保存。
