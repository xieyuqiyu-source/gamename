## 3. PRISM WARDEN Boss

### 3.1 核心参数

- 尺寸：`166 × 54`。
- 最大耐久：12。
- 阶段：3。
- 每阶段核心耐久：4。
- 阶段移动速度：`88 / 126 / 172` 逻辑像素每秒。
- 三阶段分别使用独立护盾砖阵。

### 3.2 战斗循环

```text
击碎当前护盾节点
  → 护盾破裂、核心暴露
  → 小球或激光攻击核心
  → 累计造成 4 点核心伤害
  → 下一阶段居中、加速并重构护盾
  → 第三阶段核心归零
  → 正式胜利结算
```

关键规则：

- 护盾存在时，核心会反弹小球但不会扣除耐久。
- 护盾节点全部清空后不会提前胜利，只会进入核心暴露状态。
- 小球和双轨激光都可以对暴露核心造成伤害。
- 全局命中冷却避免多球或双激光在同一瞬间跳过阶段。
- 核心降至 8、4 点时分别进入第二、第三阶段。
- 核心周期性释放可视脉冲；阶段越高，移动和脉冲节奏越快。
- 最终核心降到 0 后才调用原有胜利、三星、奖励和存档流程。

### 3.3 视觉反馈

- Canvas 顶部显示 `BOSS P阶段 / 护盾节点或核心耐久`。
- 核心使用菱形能量体、双翼、辉光、护盾椭圆和移动轨迹。
- 护盾破裂使用金色爆发、全屏闪光和增强震屏。
- 阶段重构使用大量章节色粒子、脉冲波纹和状态文案。
- 桌面操作终端显示代号、阶段、血条和 `SHIELDED / ATTACK` 状态。
- 手机不增加额外纵向卡片，复用紧凑状态行显示阶段和护盾/核心值。
