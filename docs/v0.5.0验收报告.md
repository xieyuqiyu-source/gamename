## 1. 本阶段目标

本阶段把 v0.4.0 的单关纵切扩展为可持续制作完整游戏的战役与成长骨架：

- 建立 4 章 20 关的数据驱动模型。
- 建立战役选关、顺序锁定、关卡预览和部署流程。
- 建立 7 类永久强化及购买、持久化和免费重置。
- 建立累计星级奖励和一次性领取记录。
- 将关卡参数与永久强化真正装载到 GameEngine。
- 将正式存档提升到 v2，并安全迁移旧档。
- 保持桌面和手机双端首屏可用，继续执行真实 Chrome 验收。

- 胶囊概率加成和晶币倍率均进入实际掉落/接取流程。

### 2.3 星级累计奖励

累计达到 `3 / 6 / 9 / 12 / 15 / 30 / 45 / 60` 星时，分别可以领取 `30 / 45 / 60 / 80 / 120 / 180 / 260 / 400` 晶币。

领取记录写入 `campaign.claimedStarRewards`，相同奖励不能重复领取。自动化验证确认 3 星奖励增加 30 晶币，刷新后领取状态保持。

### 2.4 存档 v2

正式存档键仍为 `gamename:save`，版本提升到 `saveVersion = 2`。新增累计星级奖励领取记录，并在迁移后的档案中记录来源版本。

兼容策略：

- v1 正式存档自动迁移到 v2。
- 原有晶币、关卡星级、最高分、连击、挑战次数、通关次数、升级与设置全部保留。
- 更早的 `gamename:game` 快照仍通过既有迁移入口转换。
- 损坏 JSON 仍先备份原始字符串，再建立安全新档。
