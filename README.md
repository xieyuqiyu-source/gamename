# Neon Breaker（项目代号）

一款使用 `Vue 3 + Pinia + Vite + Canvas 2D` 开发的纯前端竖屏霓虹打砖块游戏。项目将持续迭代到完整版本，目标包含 20 个主线关卡、4 个 Boss、5 种即时道具、金币与永久升级、无尽模式，以及电脑和手机双端操作。

当前版本：`v0.1.0` 项目与文档基线。正式游戏名称将在后续确定。

## 玩法目标

- 保留经典 Breakout 的挡板、小球和砖块核心玩法。
- 通过拖尾、粒子、碎裂、闪光、波纹、命中停顿和震屏强化击打爽感。
- 击碎砖块会掉落即时道具与金币。
- 主线共 20 关，每 5 关一个 Boss。
- 通关第 10 关后解锁无尽模式。
- 不使用背景音乐和音效，所有反馈依靠画面表达。

## 技术栈

- Vue 3：界面、菜单、HUD 和弹窗
- Pinia：存档、进度、升级和全局状态
- Canvas 2D：游戏实体、碰撞、粒子和特效
- Vite：开发服务器和生产构建
- localStorage：纯前端本地存档
- pnpm：依赖管理

## 环境要求

- Node.js 20.19+ 或 22.12+
- pnpm 10+
- 支持 Canvas 2D 的现代浏览器

## 本地运行

安装依赖：

```bash
pnpm install
```

启动开发服务器：

```bash
pnpm dev
```

访问：

```text
http://localhost:5173/
```

生产构建：

```bash
pnpm build
```

预览构建：

```bash
pnpm preview
```

## 当前模板能力

- 已注册 Pinia 根状态。
- Pinia 状态可以持久化到 localStorage。
- 已提供响应式 `960 × 540` 模板 Canvas，正式游戏阶段会切换为 `540 × 960` 竖屏逻辑坐标。
- 按 `F` 可以切换全屏。
- `window.render_game_to_text()` 可供自动测试读取当前状态。
- `window.advanceTime(ms)` 可供自动测试确定性推进时间。

## 存档说明

游戏最终使用 `gamename:save` 作为正式存档键。当前 `v0.1.0` 仍保留模板 Store 持久化；进入正式存档阶段后会提供版本迁移。
