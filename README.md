# Neon Breaker（项目代号）

一款使用 `Vue 3 + Pinia + Vite + Canvas 2D` 开发的纯前端竖屏霓虹打砖块游戏。项目持续按阶段迭代，最终目标包含 20 个主线关卡、4 个 Boss、5 种即时道具、金币与永久升级、无尽模式，以及电脑和手机双端操作。

当前版本：`v0.3.0` 爽感与道具系统。正式游戏名称将在后续确定。

当前里程碑进度：`3 / 9` 个开发阶段完成，约 `33%`。

## 当前可玩内容

- `540 × 960` 竖屏霓虹战场
- 经典挡板、小球和砖块反弹玩法
- 51 块砖组成的第 1 关“初次折射”测试布局
- 一击破碎的普通砖
- 需要两次命中的紫色强化砖
- 3 条生命、掉球重置、胜利与失败
- 键盘、鼠标和触摸控制
- 暂停、继续、重新挑战和全屏
- 基础粒子、闪光、拖尾、辉光和震屏
- 命中停顿、冲击波、浮动分数、强化粒子池和连击反馈
- 多球分裂、挡板扩展、穿透光球、时流减速和双轨激光
- 最多 12 颗球，不同道具同时生效，同类计时道具刷新时间
- 晶币散落、重力弹跳、挡板磁吸和本地持有数量
- 发光胶囊掉落、挡板接取、漏接消失和掉球统一清空
- 连击倒计时、分数倍率和最高连击
- 桌面三栏 UI 和手机单屏 UI
- Pinia 状态摘要与最高分本地保存

当前版本尚未加入正式关卡结算、永久升级、Boss 和无尽模式。

## 操作说明

### 电脑

- 鼠标移动：控制挡板
- `A` / `D`：控制挡板
- `←` / `→`：控制挡板
- 点击战场或 `Space`：发球
- 激光生效时点击战场或 `Space`：发射双轨激光
- `P` 或 `Esc`：暂停/继续
- `F`：切换全屏

### 手机

- 在战场内左右拖动：控制挡板
- 轻触战场或底部按钮：发球
- 页面底部按钮：暂停或继续

## 技术栈

- Vue 3：界面、HUD 和操作按钮
- Pinia：生命、得分、关卡进度和设置摘要
- Canvas 2D：游戏实体、碰撞、砖块和特效
- Vite：开发服务器和生产构建
- localStorage：最高分与后续正式存档
- pnpm：依赖管理

## 环境要求

- Node.js 20.19+ 或 22.12+
- pnpm 10+
- 支持 Canvas 2D 的现代浏览器

## 本地运行

```bash
pnpm install
pnpm dev
```

访问：

```text
http://localhost:5173/
```

生产构建：

```bash

```

## 测试接口

- `window.render_game_to_text()`：读取模式、生命、得分、晶币、连击、挡板、小球、砖块、掉落物、道具和特效状态。
- `window.advanceTime(ms)`：按固定时间步确定性推进游戏。
- 开发环境提供 `window.__NEON_BREAKER_TEST__`，用于胜负和物理边界回归。
